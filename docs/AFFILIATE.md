# DecodedSix — Affiliate Program Registry

## Active After Gate 2 (AdSense Approved)

### Amazon Associates
Program: Amazon Associates
Network: Direct
Commission: 3-4% on gaming hardware, accessories
Cookie: 24 hours (90 days if added to cart)
Tracking: amazon.com/associates
Best placements:
  - "Best controllers for GTA 6" guides
  - Peripheral comparison articles
  - Setup/gear recommendation content
Notes: Highest volume potential due to trust factor.
       Use for any product recommendation under $200.

## Placement Rules

### Text Links
- Natural anchor text only — never "click here" or "buy now"
- One affiliate link per 500 words max
- Always disclosed: "(affiliate link)" in parentheses

### Comparison Tables
- Build after Gate 7 (once we know what converts)
- Max 5 products per table
- Price, commission, and conversion rate tracked per slot

### Video Descriptions (YouTube)
- One affiliate link per video
- First or second line of description only
- Always disclosed: "Affiliate link" label

## Disclosure
Every page with affiliate links includes:
"This page contains affiliate links. We may earn a commission
 at no extra cost to you if you purchase through our links."
Place in site footer and at top of any heavy-affiliate article.

## Tracking
All clicks tracked via affiliate_clicks table.
All conversions tracked via affiliate_conversions table.
DS-AFF agent reads both tables for optimization.
Revenue per placement reported in dashboard revenue panel.
